Voltajes

Primero El voltaje se define como la magnitud encargada de establecer la diferenciación de potencial eléctrico que existe entre dos puntos. Es por esto que también se le conoce como tensión eléctrica o diferencia de potencial eléctrica y en las embarcaciones se trabaja con 380 voltios, 220 voltios en corriente alterna y 24 y 12 voltios corriente directa

¿Qué alimenta cada voltaje?

380 voltios se utiliza para las bombas de achique para baldear cubiertas y lo principal para achicar agua en caso de inundación

220 voltios son utilizados para iluminacion, bombas de agua dulce, circuitos de enchufes, sistema de cargas para las baterias.

24 voltios son utilizados para luces de navegacion, sistemas de comunicaciones, sistemas de alarmas de incendio. alarmas de inundacion, luces de emergencia, bombas de achique de emergencia, bombas de agua dulce en caso de emergencia cuando el sistema de generación 380 - 220 voltios no esta operativo

12 voltios se emplea para los sistemas de navegacion camaras de seguridad.
